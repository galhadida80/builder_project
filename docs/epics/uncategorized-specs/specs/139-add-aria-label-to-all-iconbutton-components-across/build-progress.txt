=== IMPLEMENTATION NOTES ===

Pattern to follow for adding aria-label:

Before:
  <IconButton onClick={onClose} size="small">
    <CloseIcon />
  </IconButton>

After:
  <IconButton onClick={onClose} size="small" aria-label="Close dialog">
    <CloseIcon />
  </IconButton>

Common aria-label values:
- Close buttons: "Close dialog", "Close drawer", "Close details"
- Edit buttons: "Edit [entity]" (e.g., "Edit material", "Edit contact")
- Delete buttons: "Delete [entity]" (e.g., "Delete material", "Delete contact")
- View buttons: "View details"
- Navigation: "Back to projects", "Open notifications", "Open profile menu"
- Toggles: "Toggle theme", "Show password", "Hide password"
- Menu: "Open project menu", "Open menu"

Dynamic aria-labels (password toggles):
  aria-label={showPassword ? "Hide password" : "Show password"}
