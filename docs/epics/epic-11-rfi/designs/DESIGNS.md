# Epic 11: RFI System - Design Screens

Project ID: `1875263988375601203` (BuilderOps Dashboard)

## Screens
1. **RFI List Page** - Data table with RFI numbers, status badges, priority chips, filters
2. **RFI Detail Page** - Email thread view with responses, attachments, reply input
3. **RFI Form Dialog** - Create form with to_email, subject, category, priority, question fields

## Key UI Elements
- Status badges: open (blue), answered (green), closed (gray), overdue (red)
- Priority chips: low/medium/high/urgent
- Email thread chronological view
- Dashboard widget with RFI statistics
