# BUG-524: No unsaved changes warning when closing dialog

## Severity
Medium

## Category
Projects

## Steps to Reproduce
1. Open "New Project" dialog
2. Fill in some fields
3. Click Cancel or X to close
4. Observe if warning shown

## Expected Result
Should warn user:
- "You have unsaved changes"
- Confirm before closing
- Option to continue editing
- Prevent accidental data loss

## Actual Result
Dialog closes immediately
All entered data lost
No warning to user
Frustrating user experience
