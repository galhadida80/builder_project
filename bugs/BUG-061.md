# BUG-061: No file upload validation (size, type, virus scan)

## Severity
Critical

## Category
Files

## Steps to Reproduce
1. Try to upload very large file (> 100MB)
2. Try to upload executable file (.exe, .sh)
3. Check validation

## Expected Result
Should validate:
- Max file size (10-50MB reasonable)
- Allowed file types only
- Virus scanning
- Clear error messages

## Actual Result
Likely no frontend validation
Could upload malware
Server could crash with huge files
Security vulnerability
