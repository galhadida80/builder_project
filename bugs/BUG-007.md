# BUG-007: No date picker widget for Start Date and End Date fields

## Severity
High

## Category
Projects

## Steps to Reproduce
1. Navigate to Projects page
2. Click "New Project"
3. Click on Start Date field

## Expected Result
Should open a calendar/date picker widget allowing user to select date visually

## Actual Result
Just a plain textbox that accepts any text input. User must manually type date in unknown format
No guidance on expected date format (YYYY-MM-DD? MM/DD/YYYY? DD/MM/YYYY?)
