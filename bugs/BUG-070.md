# BUG-070: No logout functionality visible

## Severity
High

## Category
Projects

## Steps to Reproduce
1. Look for logout button/option
2. Check user menu, settings, sidebar

## Expected Result
Should have logout option:
- In user dropdown menu
- Clear "Sign Out" or "Logout" button
- Confirmation before logout
- Clears session properly

## Actual Result
No logout option visible
Users cannot log out
Security concern
Must close browser to end session
