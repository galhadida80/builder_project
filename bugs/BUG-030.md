# BUG-030: No filtering by project status

## Severity
Medium

## Category
Projects

## Steps to Reproduce
1. Navigate to Projects page
2. Look for filter controls
3. Try to filter by active/completed/archived

## Expected Result
Should have filter options:
- Show All
- Active only
- Completed only
- Archived
- Multi-select filters

## Actual Result
No status filtering available
All projects shown together
User cannot focus on specific project states
Cluttered view with mixed statuses
