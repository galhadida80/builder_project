# BUG-009: End Date can be before Start Date

## Severity
High

## Category
Projects

## Steps to Reproduce
1. Navigate to Projects page
2. Click "New Project"
3. Enter "2020-01-01" in Start Date
4. Enter "2019-01-01" in End Date (before start date)
5. Try to submit

## Expected Result
Should show validation error: "End Date must be after Start Date"

## Actual Result
System accepts End Date before Start Date, creating logically invalid project timeline
