# BUG-049: No phone number formatting or validation

## Severity
Medium

## Category
Contacts

## Steps to Reproduce
1. Add contact
2. Enter phone number
3. Observe formatting

## Expected Result
Should format phone numbers:
- Auto-format as user types
- Accept international formats
- Validate phone number format
- Store in consistent format

## Actual Result
Free text field
Users enter inconsistent formats:
- (555) 123-4567
- 555-123-4567
- 5551234567
- +1-555-123-4567
Poor data quality
