# BUG-017: No error message shown when material creation fails with large quantity

## Severity
High

## Category
Materials

## Steps to Reproduce
1. Navigate to a project's Materials tab
2. Click "Add Material"
3. Enter Material Name: "Concrete Mix"
4. Enter Quantity: 999999999999999999
5. Click Add Material

## Expected Result
Should show user-friendly error message:
- "Quantity value is too large"
- "Please enter a reasonable quantity"
- Toast notification with error details

## Actual Result
Dialog stays open, no error message shown to user
Only visible in console: "Failed to create material: AxiosError"
User has no idea what went wrong
