# BUG-031: Equipment status cannot be changed from Draft

## Severity
High

## Category
Equipment

## Steps to Reproduce
1. Add equipment item (status defaults to "Draft")
2. Look for way to change status to "Active" or other states
3. Try to update equipment status

## Expected Result
Should be able to change equipment status:
- Draft → Active
- Active → Maintenance
- Active → Decommissioned
- Status change dropdown or edit function

## Actual Result
No visible way to change equipment status
Equipment stuck in "Draft" forever
Cannot track equipment lifecycle
Critical workflow missing
