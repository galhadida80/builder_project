# BUG-044: Meeting location field is free text, no address validation

## Severity
Low

## Category
Meetings

## Steps to Reproduce
1. Click "Schedule Meeting"
2. Enter location
3. Observe field type

## Expected Result
Should have:
- Address autocomplete
- Link to map
- Or preset locations (Site, Office, Virtual)
- "Add to calendar" generates proper location

## Actual Result
Simple text field
No address validation
Cannot link to maps
Users might enter inconsistent locations
