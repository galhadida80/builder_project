# BUG-042: No attendee selection for meetings

## Severity
Critical

## Category
Meetings

## Steps to Reproduce
1. Click "Schedule Meeting"
2. Look for attendee selection
3. Check if contacts can be invited

## Expected Result
Should have attendee management:
- Select from project contacts
- Add email addresses
- Required/Optional attendees
- See who's invited

## Actual Result
No attendee selection visible in form
Button says "Schedule & Send Invites" but to whom?
Cannot invite people to meetings
Core meeting functionality missing
