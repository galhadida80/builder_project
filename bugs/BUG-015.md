# BUG-015: Equipment Type not displayed in table

## Severity
High

## Category
Equipment

## Steps to Reproduce
1. Navigate to a project's Equipment tab
2. Click "Add Equipment"
3. Fill in Equipment Name: "Excavator CAT 320"
4. Select Equipment Type: "Heavy Machinery"
5. Fill in Manufacturer: "Caterpillar"
6. Fill in Model Number: "320GC"
7. Click Add Equipment

## Expected Result
Equipment table should show:
- Name: Excavator CAT 320
- Type: Heavy Machinery
- Manufacturer: Caterpillar
- Model: 320GC

## Actual Result
Table shows:
- Name: Excavator CAT 320
- Type: - (empty/missing)
- Manufacturer: Caterpillar
- Model: - (empty/missing)

Equipment Type and Model Number are not displayed even though they were entered
