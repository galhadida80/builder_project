# BUG-201: Dialog forms not keyboard accessible (Tab key navigation broken)

## Severity
High

## Category
Projects

## Steps to Reproduce
1. Open "New Project" dialog
2. Try to navigate fields using Tab key
3. Try to submit using Enter key

## Expected Result
Tab key should navigate through fields in order
Enter key should submit form
Escape key should close dialog
Full keyboard accessibility

## Actual Result
Tab navigation likely broken
Cannot submit with keyboard
Not accessible for keyboard users
WCAG compliance failure
