# BUG-032: No file attachment capability for equipment

## Severity
Medium

## Category
Equipment

## Steps to Reproduce
1. Add equipment item
2. Look for way to attach files (manuals, inspection reports, photos)
3. Check equipment details

## Expected Result
Should allow file attachments:
- User manuals PDF
- Inspection reports
- Photos of equipment
- Maintenance records
- "Attach File" button visible

## Actual Result
No file attachment capability
Cannot store important equipment documentation
Users must track files separately
Missing core feature for equipment management
