# BUG-050: No way to mark primary contact for project

## Severity
Medium

## Category
Contacts

## Steps to Reproduce
1. Add multiple contacts to project
2. Try to mark one as "Primary Contact"
3. Look for primary designation

## Expected Result
Should be able to:
- Mark one contact as primary
- Display primary contact prominently
- Quick access to primary contact
- "Make Primary" button/toggle

## Actual Result
All contacts treated equally
No way to identify main contact
Must remember who is primary
Poor UX for contact management
