# BUG-011: Project details page shows no content - tabs not working

## Severity
Critical

## Category
Projects

## Steps to Reproduce
1. Navigate to Projects page
2. Click on a project's "View Details"
3. Observe the page

## Expected Result
Should show:
- Project overview/details in a default tab
- Visible tab content area below the tabs

## Actual Result
Only tabs are visible but no content is shown below them
No indication which tab is active
User sees empty page with no information about the project
