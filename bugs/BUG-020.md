# BUG-020: Contacts endpoint returns network error - ERR_FAILED

## Severity
Critical

## Category
Contacts

## Steps to Reproduce
1. Navigate to a project's Contacts tab

## Expected Result
Contacts should load successfully or show proper error message

## Actual Result
Multiple console errors:
- "Access to XMLHttpRequest... ERR_FAILED"
- "Failed to load contacts: AxiosError: Network Error"
Page shows "No contacts found" but it's actually an error loading
User cannot distinguish between "no contacts" and "error loading contacts"
