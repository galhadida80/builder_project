## Severity
High

## Category
Areas

## Steps to Reproduce
1. Create/edit area
2. Try to enter progress as -10%
3. Submit

## Expected Result
Should validate:
- "Progress cannot be negative"
- Block negative values
- Only accept 0-100 range

## Actual Result
Likely accepts negative values
Invalid progress data
Nonsensical progress tracking
Validation missing
