# BUG-033: Equipment table not sortable by clicking column headers

## Severity
Medium

## Category
Equipment

## Steps to Reproduce
1. Navigate to Equipment tab
2. Click on column headers (Name, Type, Manufacturer, etc.)
3. Try to sort the table

## Expected Result
Clicking column headers should sort:
- First click: ascending
- Second click: descending
- Visual indicator (arrow) showing sort direction

## Actual Result
Column headers not clickable for sorting
No way to organize equipment list
Difficult to find specific equipment in long lists
Poor table UX
