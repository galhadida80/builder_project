# BUG-066: Project selector shows "Select a project" but doesn't work

## Severity
High

## Category
Projects

## Steps to Reproduce
1. Look at header "Select a project" dropdown
2. Click on it
3. Try to select different project

## Expected Result
Should show list of user's projects
Clicking switches active project
Updates all tabs to show selected project data

## Actual Result
Dropdown likely not functional
Cannot switch between projects
Must navigate back to projects list
Poor navigation UX
