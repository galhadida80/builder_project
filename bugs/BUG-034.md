# BUG-034: No bulk operations for equipment

## Severity
Medium

## Category
Equipment

## Steps to Reproduce
1. Navigate to Equipment tab with multiple items
2. Try to select multiple equipment items
3. Look for bulk action options

## Expected Result
Should support bulk operations:
- Select multiple checkboxes
- Bulk delete
- Bulk status change
- Export selected items
- "Select All" option

## Actual Result
No checkbox selection
No bulk operations
Must process each item individually
Tedious for managing many items
