# BUG-029: No sorting options for project list

## Severity
Medium

## Category
Projects

## Steps to Reproduce
1. Navigate to Projects page
2. Look for sorting controls
3. Try to sort by name, date, status

## Expected Result
Should have sorting dropdown:
- Sort by Name (A-Z, Z-A)
- Sort by Date Created (Newest, Oldest)
- Sort by Status
- Sort by Project Code

## Actual Result
No sorting controls visible
Projects appear in arbitrary order
User cannot organize view by preference
Difficult to find projects in large lists
