# BUG-013: Equipment Name accepts whitespace-only input

## Severity
High

## Category
Equipment

## Steps to Reproduce
1. Navigate to a project's Equipment tab
2. Click "Add Equipment"
3. Enter only spaces in Equipment Name field
4. Click Add Equipment

## Expected Result
Should show validation error "Equipment Name cannot be empty or contain only whitespace"

## Actual Result
Still shows "Equipment Name is required" but field appears filled with spaces
Same issue as Project Name field
