# BUG-004: No user feedback when project creation fails

## Severity
High

## Category
Projects

## Steps to Reproduce
1. Navigate to Projects page
2. Click "New Project"
3. Fill in Project Name and Project Code
4. Click Create Project (when backend returns 422 error)

## Expected Result
Should show user-friendly error message like:
- "Failed to create project. Please check your inputs"
- "Server error occurred. Please try again"
- Toast notification with specific error details

## Actual Result
Dialog stays open, no error message shown to user. Only visible in console: "Failed to create project: AxiosError"
User has no idea what went wrong or what to do next
