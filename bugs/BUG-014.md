# BUG-014: HTML/Script tags accepted in Equipment Name - XSS vulnerability

## Severity
Critical

## Category
Equipment

## Steps to Reproduce
1. Navigate to a project's Equipment tab
2. Click "Add Equipment"
3. Enter "<img src=x onerror=alert('xss')>" in Equipment Name
4. Click Add Equipment

## Expected Result
Should sanitize or reject HTML/script tags

## Actual Result
HTML tags are accepted and displayed in the equipment list
Creates XSS security vulnerability
The equipment row shows: "<img src=x alert('xss')>"
