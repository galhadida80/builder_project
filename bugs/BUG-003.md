# BUG-003: HTML/Script tags accepted in Project Name field

## Severity
Critical

## Category
Projects

## Steps to Reproduce
1. Navigate to Projects page
2. Click "New Project"
3. Enter "<script>alert('xss')</script>" in Project Name field
4. Enter "TEST-001" in Project Code
5. Click Create Project

## Expected Result
Should either:
- Strip HTML tags before saving
- Show validation error
- Encode the input properly

## Actual Result
System accepts HTML/script tags in project name, creating potential XSS vulnerability
