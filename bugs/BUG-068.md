# BUG-068: Notification bell shows "3" but no notifications visible

## Severity
High

## Category
Projects

## Steps to Reproduce
1. Observe notification bell with "3" badge
2. Click on notification bell
3. Try to view notifications

## Expected Result
Should show:
- List of 3 notifications
- Notification details
- Mark as read functionality
- Notification settings

## Actual Result
Badge shows "3" but clicking does nothing
Cannot view notifications
Misleading UI
Feature incomplete or broken
