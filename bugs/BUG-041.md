# BUG-041: Meeting time validation - End Time can be before Start Time

## Severity
High

## Category
Meetings

## Steps to Reproduce
1. Click "Schedule Meeting"
2. Set Date to 2026-02-01
3. Set Start Time to 14:00
4. Set End Time to 13:00 (before start)
5. Submit

## Expected Result
Should show validation error:
- "End time must be after start time"
- Prevent illogical meeting times

## Actual Result
Likely accepts end time before start time
Creates invalid meeting schedule
Confusing for attendees
Logic error in validation
