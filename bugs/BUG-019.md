# BUG-019: No frontend validation for required meeting fields

## Severity
Critical

## Category
Meetings

## Steps to Reproduce
1. Navigate to a project's Meetings tab
2. Click "Schedule Meeting"
3. Leave all fields empty (including required fields marked with *)
4. Click "Schedule & Send Invites"

## Expected Result
Should show validation errors:
- "Meeting Title is required"
- "Date is required"
- "Start Time is required"
- "End Time is required"

## Actual Result
No frontend validation shown
Attempts to submit to backend, which fails
No error message shown to user, just console error
User has no feedback about what went wrong
