# BUG-022: No loading state when creating project

## Severity
Medium

## Category
Projects

## Steps to Reproduce
1. Click "New Project"
2. Fill in all required fields with valid data
3. Click "Create Project"
4. Observe the UI during submission

## Expected Result
Should show loading indicator:
- Spinner on button
- "Creating..." text
- Disabled form fields
- Visual feedback that action is in progress

## Actual Result
No loading state shown
User doesn't know if click registered
On slow connection, user might click multiple times
