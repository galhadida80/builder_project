# BUG-062: No upload progress indicator for large files

## Severity
Medium

## Category
Files

## Steps to Reproduce
1. Upload large file (10MB+)
2. Observe upload process

## Expected Result
Should show:
- Progress bar (0-100%)
- Upload speed
- Time remaining
- Ability to cancel upload

## Actual Result
No progress indicator
User doesn't know if upload is working
Cannot tell if it froze
Frustrating UX for large files
