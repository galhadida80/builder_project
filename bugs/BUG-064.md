# BUG-064: No file version control

## Severity
Medium

## Category
Files

## Steps to Reproduce
1. Upload file "blueprint_v1.pdf"
2. Upload updated "blueprint_v2.pdf"
3. Check if versions are tracked

## Expected Result
Should have version control:
- Track all versions
- View version history
- Restore previous versions
- Compare versions

## Actual Result
No version tracking
Files overwritten
Cannot recover old versions
Lost data risk
