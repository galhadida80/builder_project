# BUG-028: No keyboard shortcuts for common actions

## Severity
Low

## Category
Projects

## Steps to Reproduce
1. Navigate to Projects page
2. Try pressing common shortcuts:
   - Ctrl/Cmd + N (New Project)
   - Ctrl/Cmd + F (Focus search)
   - Escape (Close dialogs)

## Expected Result
Common keyboard shortcuts should work
Power users expect:
- N or Ctrl+N for New
- / or Ctrl+F for search
- Esc to close dialogs

## Actual Result
No keyboard shortcuts implemented
Users must use mouse for all actions
Slower workflow for power users
